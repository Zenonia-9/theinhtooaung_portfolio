## Problem

Odoo default addresses are too generic for Myanmar businesses and do not support the full local administrative address hierarchy.

## Solution

I built an Odoo 19 localization addon suite that adds Myanmar State or Region, District, Township, Town, Ward or Village Tract, Postal Code, and P-Code support.

## Key Features

- Myanmar administrative address hierarchy
- Cascading address dropdowns
- MIMU P-Code auto-fill
- Myanmar language address support
- Contact address extension
- CRM lead integration
- HR employee private address integration
- Full Myanmar address computation

## Technical Highlights

- Custom Odoo address models
- CSV master data import
- Many2one cascading domains
- Computed address fields
- Contact, CRM, and HR model extensions
- Township-level geolocation support

- Add address search optimization
- Add more report templates
