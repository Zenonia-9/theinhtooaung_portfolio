## Problem

Odoo standard workflows do not provide a complete consignment sales process for shop-based stock movement, partial transfers, backorders, settlements, commission billing, and returns.

## Solution

I built a custom Odoo 19 module that connects Inventory, Accounting, Contacts, Products, and Reports into one complete consignment workflow.

## Key Features

- Consignment shop setup
- Consignment order workflow
- Automatic stock transfer creation
- Partial transfer and backorder tracking
- Dedicated consignment transfers menu
- Settlement workflow
- Customer invoice generation
- Commission vendor bill generation
- Return workflow for unsold goods
- A4/A5 QWeb PDF reports

## Technical Highlights

- Custom Odoo models
- Stock picking integration
- Backorder propagation
- Computed transfer status
- Invoice and vendor bill creation
- QWeb reporting
- Multi-company access rules
